# ✅ Pre‑Lab Checklist

Please confirm the following before the workshop:

## Hardware & OS
- Windows 10/11, macOS, or Linux (x86‑64)
- ≥ 8 GB RAM recommended

## Software
- Python 3.10–3.12 (`python --version`)
- Git (`git --version`)
- Optional: VS Code + Python extension

## Network
- Egress allowed to: `https://models.inference.ai.azure.com`
- If behind a proxy: you know the proxy URL and have a root CA file path if TLS inspection is used

## Credentials
- A **GitHub token** with **Models** inference access (per your enterprise policy)

## Local setup
- Ability to create and activate a Python virtual environment
- Optional: access to an internal share/USB if you plan to preload wheels
